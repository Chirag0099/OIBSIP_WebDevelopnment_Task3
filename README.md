# OIBSIP - Web Development & Designing - Task 3
## To-Do Web Application

## Objective
To develop a task management application that helps users organize daily activities.

## Features
- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks as completed
- Local Storage support
- Responsive interface

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Local Storage API

## Steps Performed
1. Designed the application interface.
2. Implemented CRUD operations for tasks.
3. Added Local Storage for data persistence.
4. Tested all functionalities.

## Outcome
Developed a fully functional To-Do application with persistent task storage.
